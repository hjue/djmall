项目截图

## 安装

前端

```
cd frontend
pnpm install 

```

服务端

```
cd backend
pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
```

## 运行

服务端

```
cd backend
python manage.py runserver
```



前端

```
cd frontend
pnpm run dev
```
